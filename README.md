### Getting started

#### Install snapx dotnet 

- `dotnet tool update snapx -g`
- `git clone https://github.com/fintermobilityas/snapx.demoapp.git`
- `cd snapx.demoapp`
- `snapx`

##### Restore offline/web installers

- `snapx restore -i`
